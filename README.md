# Team 1: Fare la spesa seguendo una lista

- **guardare nel frigo**
  - **FINCHÉ il prodotto è assente**
    - aggiungi elemento alla lista
  - scrivere le crocchette per Ricky

- **andare al supermercato**
  - prendere il portafogli e il coupon
  - prendere le buste della spesa
  - prendere la lista
  - **SE c'è il sole**
    - usa la bicicletta
  - **ALTRIMENTI**
    - usa la macchina

- **fare spesa**
  - prendere il carrello
  - **FINCHÉ la lista non è completamente spuntata**
    - andare nella sezione
    - aggiungere prodotto al carrello
    - spuntare lista per ogni prodotto preso
      - **SE ci sono le crocchette che piacciono a Ricky**
        - aggiungi al carrello
  - andare alla cassa per pagare
    - **SE il valore dei coupon copre la spesa totale**
      - usare il coupon
      - acquistare gli snack extra
    - **ALTRIMENTI**
      - usare il coupon e il portafogli

  - tornare a casa
    - sistemare la spesa
    - versare le crocchette nella ciotola di Ricky
    - **SE ho comprato gli snack**
      - mi gusto lo snack
    - **ALTRIMENTI**
      - piango in un angolo abbracciando le ginocchia
